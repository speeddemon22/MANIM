# Coupled Oscillators & Normal Modes 

**Manim animations** of advanced physics from Oscillations & Waves course.

## Scenes 📈

- **`scene2.py`** – **Coupled oscillators** (energy transfer between masses)
- **`scene3.py`** – **Normal modes** (symmetric/antisymmetric oscillation patterns)
- **`scene4.py`** – **General motion** as superposition of normal modes
- **`MATHTEX.py`** – LaTeX equation typesetting

## Run locally 🖥️

```bash
pip install manim
manim scene2.py Scene2ClassName -p  # preview window
# or -pqh for 1080p video
```
